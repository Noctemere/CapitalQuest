# CapitalQuest

CapitalQuest is a sophisticated financial investment simulation game built with Next.js. Players navigate the world of finance, managing a portfolio of various assets to reach an ambitious net worth goal.

## üéÆ Features

- **Real-time Portfolio Management**: Buy and sell various asset classes including Stocks, Bonds, Funds, and Crypto.
- **Dynamic Time Simulation**: Watch your investments grow (or shrink) as game time passes. Control the simulation speed to see long-term results in minutes.
- **Risk & Return Dynamics**: Each asset has unique risk profiles and return rates. Balance your portfolio between safe bonds and volatile crypto.
- **Win Condition Tracking**: Monitor your total net worth and ROI as you strive to reach your financial goals.
- **Tax Simulation**: Features capital gains tax and withdrawal tax considerations for a more realistic experience.

## üöÄ Getting Started

First, ensure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

Clone the repository:

```bash
git clone https://github.com/your-username/capital-quest.git
cd capital-quest
```

Install dependencies:

```bash
npm install
```

### Running the Development Server

Start the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## üïπÔ∏è How to Play

1. **Start Small**: You begin with a starting balance (default $1,000).
2. **Research Assets**: Look at the available assets, their base return rates, and volatility levels.
3. **Invest**: Select an asset and enter the amount you wish to invest.
4. **Time Management**: Use the speed controls (1x, 2x, 5x, 10x) to advance time and see how your investments perform.
5. **Rebalance**: Sell assets that have performed well to lock in profits or reallocate funds to other opportunities.
6. **Goal**: Reach your target net worth (default $1,000,000,000,000) to win the game!
